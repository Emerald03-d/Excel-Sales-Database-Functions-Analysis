# 📊 Excel Sales Database Functions Analysis

## 📌 Project Overview

This project demonstrates the use of Microsoft Excel database and conditional functions to analyze sales data and generate dynamic reports. Users can interact with the workbook by selecting a **Product**, **Region**, and **Department** from drop-down lists, and all calculations update automatically based on the selected criteria.

The project showcases practical Excel techniques commonly used in business reporting and data analysis.

---

## 🎯 Objectives

This workbook is designed to:

* Determine employee bonus eligibility based on total sales.
* Calculate Total Bill Invoice, Total Sales, Average Sales, Highest Sales, and Lowest Sales for a selected product.
* Count records using single and multiple criteria.
* Perform sales analysis using multiple criteria (Product, Region, and Department).
* Implement Data Validation to create interactive drop-down lists.
* Build a dynamic worksheet that automatically recalculates results based on user selections.
